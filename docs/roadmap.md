- Approval flow
- Receiving workflow
- Cost tracking

- Agent-facing APIs
- Recommendation storage
- Policy engine
- Approval orchestration

## Stage 5: Automation

- Supplier integrations
- Budget-aware ordering
- Automatic reorder triggers
- Forecast refinement
